Dropdown

The select element defines a drop-down list.Here is the example of select element used in our applications form,

Here is the html and css code for dropdown used in our application,

                                
        
        <form>
        <label for="example">Choose an item:</label>
        <select id="example" name="example">
        <option value="example">item1></option>
        <option value="example">item2></option>
        <option value="example">item3></option>
        </select>
        </form>
