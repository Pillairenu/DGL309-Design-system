Input Type - Checkbox

Here is the example of input type checkbox used in our applications form,

Input type checkbox defines a checkbox that let a user select zero or more options of a limited number of choices.



Here is the html code of our app's form input type checkbox,

    
        
    <form>
    <input type="checkbox"  name="checkbox1" value="checkbox1">
    <label for="checkbox">Example1:</label>
    <input type="checkbox"  name="checkbox2" value="checkbox2">
    <label for="checkbox">Example2:</label>
    <input type="checkbox"  name="checkbox3" value="checkbox3">
    <label for="checkbox">Example3:</label>    
    </form>
