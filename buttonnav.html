Button Navigation

The button navigation is a secondary navigation in the body of our application. It is used to move to a particular page in the application.

Here is the example of our app's button navigation,

Here is the html and css code of our app's button navigation,

                                
    <div class="button-container">                                
    <button onclick="location.href='index.html';" class="button">
    Read More..
    </button>
    </div>     
                                
                            
                
    .button-container button {
        background-color:  rgb(240, 93, 91);
        border: none;
        border-radius: 20px;
        color: white;
        padding: 15px 32px;
        text-align: center;
        text-decoration: none;
        font-size: 16px;
        
        }
        .button-container button:hover {
        background-color: #6F4E37;
        
        }
